# ❄️ Air Conditioner Electricity Cost Estimator

An interactive web application built with **Streamlit** that uses a **Random Forest Regression** machine learning model to predict real-world monthly energy consumption ($kWh$) and electricity costs for household air conditioners.

## 🚀 Features
* **ML-Driven Predictions:** Uses an AI model instead of rigid formulas to simulate dynamic compressor behavior.
* **Inverter vs. Non-Inverter Detection:** Automatically adjusts energy curves based on the appliance's CSPF efficiency score.
* **Thermodynamic Realism:** Accounts for higher initial startup loads on short runs and energy-saving throttling during long continuous use.
* **Instant Cost Breakdown:** Input your local electricity tariff ($\text{₱/kWh}$) to see localized monthly and annual expense projections.
